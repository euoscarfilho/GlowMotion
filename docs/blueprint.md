# **App Name**: GlowMotion

## Core Features:

- LED Display: Display a grid of customizable, multi-colored LEDs on the screen.
- Pattern Selection: Allow users to select from a variety of built-in light patterns.
- Animation Speed Control: Enable users to adjust the animation speed of the selected light pattern.
- Color Customization: Allow users to select the specific colors for the LEDs.
- Full-Screen Mode: Provide a full-screen mode that prevents the device screen from turning off.
- Suggest palette: AI tool generates suggested color palettes to use with the lights.

## Style Guidelines:

- Background color: Dark background (#121212) to enhance the vibrancy of LED colors.
- Primary color: Vibrant electric blue (#7DF9FF) to emulate the bright light of LEDs. 
- Accent color: Analogous vivid cyan (#7FFFD4), for control elements.
- Font: 'Inter' sans-serif font, for a clean and modern UI.
- Use simple, outlined icons for pattern selection and controls.
- Arrange controls at the bottom of the screen for easy access in full-screen mode.
- Smooth transitions between patterns and color changes.
